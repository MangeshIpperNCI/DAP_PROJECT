### 3.1 STEP I : MongoDB Run	<a id=section301></a>
** To run this project, you must have to run mongo dB server and your client should open (Robo 3T or Mongo Compass) on your PC.**
1. To run mongo server, go to your mongo dB bin path
**C:\Program Files\MongoDB\Server\4.4\bin** and run **mongod** command.



2. Make sure your client should be open and connected.
